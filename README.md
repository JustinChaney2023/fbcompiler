## Language Features

### Data Types
- int - integers (for example 5, 42)
- float - floating-point values (for example 3.14, 10.0)
  - Requires digits on both sides of the decimal point, so .5 and 5. are invalid
- bool - boolean values (true, false)

### Operators
| Type | Operators |
|------|-----------|
| Arithmetic | + - * / % |
| Comparison | == != < > <= >= |
| Logical | ! && || |
| Assignment | = |

### Control Flow
- if / else - conditional branching
- while - loops

### Built-in Functions
- sin(x) - sine function
- cos(x) - cosine function
- tan(x) - tangent function
- print(x) - outputs a value to the console and adds a newline

### Comments
// single-line comments are supported

## Building

### Prerequisites
- flex
- bison
- gcc

### Build the Lexer Test
# Generate lexer C code
flex lexer.l

# Compile test driver
gcc lex.yy.c main_test.c -o lexer_test

# Run a quick lexer test
echo "int x = 5;" | ./lexer_test

### Build the Full Compiler
# Generate parser
bison -d parser.y

# Generate lexer
flex lexer.l

# Compile compiler
gcc parser.tab.c lex.yy.c -o minicompiler -lm

## Running

### Run Interactively
./minicompiler

Then type your program directly into the terminal.
When you are finished, press Ctrl+D to signal end of input.

### Run with an Input File
./minicompiler < input_file.txt

### Run the Lexer Test on a File
./lexer_test < test.txt

## Testing

### Basic Token Test
echo "int x = 5;" | ./lexer_test

Expected output:
line  1 | INT_TYPE       | int
line  1 | IDENTIFIER     | x  => x
line  1 | ASSIGN         | =
line  1 | INT_LITERAL    | 5  => 5
line  1 | SEMICOLON      | ;

### Float Format Validation
# Valid
echo "float y = 3.14;" | ./lexer_test

# Invalid
echo "float a = .5;" | ./lexer_test
echo "float b = 5.;" | ./lexer_test

### Complex Expression
echo "z = (x && y > 3.6) || (y < 5.6);" | ./lexer_test

### Trig Functions
echo "float w = sin(x) + cos(y);" | ./lexer_test

### Sample Input File
Create test.txt:
int x = 5;
float y = 3.14;
if (x < y) {
    print(x);
}

Run:
./lexer_test < test.txt

## Tokens

| Token | Description |
|-------|-------------|
| INT_TYPE FLOAT_TYPE BOOL_TYPE | Type keywords |
| IF ELSE WHILE | Control flow |
| SIN COS TAN | Trig functions |
| PRINT | Output statement |
| TRUE FALSE | Boolean literals |
| IDENTIFIER | Variable names |
| INT_LITERAL FLOAT_LITERAL | Numeric values |
| ASSIGN EQ NE LE GE LT GT | Assignment/comparison |
| NOT AND OR | Logical operators |
| PLUS MINUS MULT DIV MOD | Arithmetic operators |
| LPAREN RPAREN LBRACE RBRACE SEMICOLON | Punctuation |

## Example Programs

### Variable Declaration and Assignment
int x = 5;
float y = 1.4;
bool z = true;

x = 6;
y = 4.3;
z = false || y >= 5.0;

### Branching
if (x > 6 || z) {
    print(a);
}
else {
    print(tan(b));
}

### Looping
while (x == 5 && y) {
    print(x);
}

### Nested Expressions
float w = sin(x) + tan(y);
z = (z && y > 3.6) || (y < 5.6) || (x > 5);

## Notes

- Whitespace is ignored, including newlines
- Identifiers must start with a letter or underscore, followed by letters, digits, or underscores
- Keywords are case-sensitive and must be lowercase
- Floating-point literals require digits on both sides of the decimal point
- Comments begin with // and continue to the end of the line
- The compiler is built with Flex and Bison, with C used for implementation support
